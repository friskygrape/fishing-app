# Fish photos — drop your images here

The app shows a placeholder until a photo exists at the exact path below.
Add a **JPG** (or change the extension in `js/data-species.js`) named exactly
as listed. Recommended: landscape, ~800×500 px, side profile of the fish.

After adding or changing photos, bump `CACHE_VERSION` in `sw.js` so the
offline cache picks them up.
